상세한 설명

풀(Pool) 기반 메모리 관리 인터페이스 및 컴포넌트 전용 풀 구현

풀을 통해 객체를 재활용함으로써 캐시 친화성과 메모리 할당/해제에 따르는 오버헤드 감소 기대. PoolBase는 풀의 기본 인터페이스를 제공하며, ComponentPool은 특정 컴포넌트 타입 T에 특화된 풀 구현. Acquire 메서드를 통해 객체를 대여받고, 스마트 포인터와 커스텀 해제자를 이용해 객체 생명주기 관리.
