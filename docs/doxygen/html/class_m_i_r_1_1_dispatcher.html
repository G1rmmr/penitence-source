이벤트를 처리하기 위한 Dispatcher 클래스

Dispatcher는 특정 이벤트 타입(EventType)에 대해 다수의 핸들러(리스너)를 등록하고, 이벤트 발생 시 해당 타입에 연관된 모든 핸들러를 호출하는 역할 수행.
