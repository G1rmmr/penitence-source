상세한 설명

ECS(Entity-Component-System) 아키텍처에서 엔티티와 컴포넌트를 관리하는 클래스

Manager는 엔티티 생성/소멸, 컴포넌트 추가/조회, 특정 컴포넌트 집합을 보유한 엔티티 조회 가능. 내부적으로 마스크와 풀을 이용해 효율적인 컴포넌트 관리와 조회 성능을 제공.
