상세한 설명

ECS(Entity-Component-System) 아키텍처에서 엔티티와 컴포넌트를 관리하는 클래스

ECSManager는 엔티티 생성/소멸, 컴포넌트 추가/조회, 특정 컴포넌트 집합을 보유한 엔티티 조회 등의 기능을 제공합니다.

  • CreateEntity, DestoryEntity로 엔티티 생명주기 관리
  • AddComponent, GetComponent, HasComponent를 통해 엔티티에 컴포넌트 추가/접근
  • Query를 통해 특정 컴포넌트 조합을 가진 엔티티 목록 검색

내부적으로 마스크(Mask)와 풀(Pool)을 이용해 효율적인 컴포넌트 관리와 조회 성능을 제공합니다.
