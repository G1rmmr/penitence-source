모든 이벤트의 기본 구조체

Event 구조체는 특정 EventType을 담고 있으며, 이를 상속받아 다양한 구체적 이벤트 확장 가능.

주의
type 멤버를 통해 이벤트 핸들러나 Dispatcher는 이벤트 종류를 판별하고 알맞은 처리 로직을 적용.
