◆ in_air

bool MIR::Gravity::in_air = false

엔티티가 공중에 있는지 여부

true일 경우 엔티티는 지면에 닿아있지 않은 상태(점프 중, 낙하 중 등)이며, 중력으로 인한 낙하 계산 수행.

false일 경우 지면에 서있거나, 더 이상 낙하나 상승이 필요 없는 상태로 간주할 수 있습니다.
