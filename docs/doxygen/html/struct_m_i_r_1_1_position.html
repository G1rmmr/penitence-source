엔티티의 위치를 나타내는 컴포넌트 구조체

Position 컴포넌트는 2차원 공간 상에서 엔티티의 위치 정보 관리. x, y 좌표를 통해 엔티티가 게임 월드 내 어디에 배치되는지를 지정할 수 있음. 다른 시스템(렌더링, 물리 연산 등)에서 이 위치 정보를 활용해 엔티티의 실제 표현이나 상호작용을 처리할 수 있음.
