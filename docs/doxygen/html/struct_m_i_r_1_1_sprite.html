상세한 설명

렌더링 가능한 스프라이트 정보를 담는 컴포넌트

이 컴포넌트는 SFML의 텍스처(sf::Texture)와 스프라이트(sf::Sprite) 관리. 엔티티에 부착함으로써 그래픽 렌더링 가능. width, height 필드는 스프라이트의 가로, 세로 크기를 저장.
